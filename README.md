# TEMU小助手 - 跨境电商管理工具

## 📋 项目概述

TEMU小助手是一款基于Electron开发的跨境电商管理桌面应用，专为TEMU平台卖家设计。通过自动化Cookie管理和多站点支持，为卖家提供便捷的店铺管理功能。

## ✨ 核心功能

### 🔐 智能登录系统
- **双层认证机制**：自动管理跨境卖货平台和TEMU站点的Cookie
- **站点自动识别**：智能识别并处理多站点跳转
- **Cookie持久化**：保存登录状态，支持快速切换
- **安全退出**：一键清除所有认证信息

### 📊 实时状态监控
- **认证状态可视化**：实时显示两层Cookie状态
- **店铺信息展示**：显示当前登录的店铺和用户信息
- **站点区域标识**：明确显示当前所在站点区域

### 🛠️ 功能模块（开发中）
- **批量上传**：批量上传商品信息
- **价格监控**：实时监控竞品价格变化
- **自动回复**：智能客服自动回复系统
- **数据导出**：一键导出店铺运营数据
- **库存预警**：库存不足自动提醒
- **促销管理**：活动和优惠券管理

## 🚀 快速开始

### 环境要求
- Node.js 14.0 或更高版本
- Windows 7/10/11 或 macOS 10.10+
- 稳定的网络连接

### 安装步骤

1. **克隆项目**
```bash
git clone [项目地址]
cd TEMU小助手
```

2. **安装依赖**
```bash
npm install
```

3. **启动应用**
```bash
npm start
```

### 首次使用

1. 启动应用后，点击"登录"按钮
2. 在弹出的窗口中登录跨境卖货平台账号
3. 登录成功后，系统会提示选择站点
4. **重要**：请选择"全球/美国"站点进入TEMU
5. 系统自动获取Cookie并关闭登录窗口
6. 返回主界面即可使用所有功能

## 🏗️ 技术架构

### 技术栈
- **框架**：Electron 33.3.1
- **前端**：HTML5 + CSS3 + JavaScript (Vanilla)
- **进程通信**：IPC (Inter-Process Communication)
- **HTTP请求**：Electron net模块
- **Cookie管理**：Electron session API

### 项目结构
```
TEMU小助手/
├── main.js           # 主进程文件
├── renderer.js       # 渲染进程文件
├── preload.js        # 预加载脚本
├── index.html        # 主界面
├── package.json      # 项目配置
├── README.md         # 项目说明
└── doc/             # 文档目录
    ├── 登录API.md   # API文档
    └── image.png    # 界面截图
```

### 核心模块说明

#### 主进程 (main.js)
- **窗口管理**：创建和管理应用窗口
- **Cookie处理**：获取、存储和恢复Cookie
- **站点切换**：处理多站点登录流程
- **IPC通信**：响应渲染进程请求

#### 渲染进程 (renderer.js)
- **UI交互**：处理用户界面事件
- **状态管理**：管理登录状态和用户信息
- **页面切换**：控制不同功能页面显示

#### 预加载脚本 (preload.js)
- **安全桥接**：在主进程和渲染进程间建立安全通信
- **API暴露**：向渲染进程暴露必要的API

### 应用配置

- **最小尺寸**：1000x650px
- **开发者工具**：默认开启（生产环境需关闭）

## 🔒 安全特性

1. **进程隔离**：主进程与渲染进程完全隔离
2. **上下文隔离**：`contextIsolation: true`
3. **禁用Node集成**：`nodeIntegration: false`
4. **Cookie加密**：敏感信息不明文存储
5. **HTTPS通信**：所有API请求使用HTTPS

## 🌟 特色功能

### 自动化流程
1. **智能Cookie管理**
   - 自动识别并保存多层Cookie
   - 站点切换时自动恢复Cookie
   - 无需重复登录

2. **新窗口拦截**
   - 防止弹出多余窗口
   - 所有跳转在当前窗口处理
   - 提升用户体验

3. **实时状态更新**
   - Cookie状态实时显示
   - 功能权限动态控制
   - 站点信息自动更新

## 📝 使用说明

### 登录流程
1. 点击登录 → 输入账号密码
2. 系统自动获取核心站点Cookie
3. 选择"全球/美国"站点
4. 自动跳转TEMU并获取站点Cookie
5. 2秒后自动关闭登录窗口

### 功能使用
- 所有功能需要完成完整登录流程
- 部分功能需要TEMU站点授权
- Cookie状态会实时显示在控制台页面

### 退出登录
- 点击右上角"退出"按钮
- 确认后清除所有Cookie
- 返回登录页面

## ⚠️ 注意事项

1. **站点选择**：目前仅支持"全球/美国"站点
2. **Cookie有效期**：长时间未使用需重新登录
3. **网络要求**：需要稳定的网络连接
4. **浏览器兼容**：使用Chromium内核，无需额外浏览器
